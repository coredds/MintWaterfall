MintWaterfall

A TypeScript waterfall chart library built on D3.js v7

v1.0.0

Financial P&L Breakdown

Classic waterfall showing how revenue flows into net income. The total bar shows the cumulative result.

Stacked Waterfall — Department Budget

Each bar can contain multiple stacks, useful for showing category breakdowns within each step.

Trend Analysis

Enable trend lines to visualize patterns across your data. Switch between linear regression, moving average, and polynomial.

Theme Gallery

MintWaterfall ships with multiple themes. Select one to see it applied in real-time.

Statistical Summary

Built-in statistical analysis: compute mean, median, variance, outlier detection, and linear regression on any dataset.
